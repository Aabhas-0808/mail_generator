# EmailCraft Pro - Smart Email Generation System

A web app that dynamically synthesizes professional and casual email drafts using client-side algorithmic rules. You input core context or keywords and it instantly builds well-structured email layouts complete with tailored subject lines and body copy.

---

## Features

- **Context-Aware Compilation**: Translates rough user prompts into clear and cohesive email messaging.
- **Tone Optimization**: Select and map communication styles between Formal, Casual, Marketing, and Urgent parameters.
- **Adjustable Text Length**: Fine-tune output density dynamically into Short, Medium, or Long layouts.
- **Clipboard Utility**: One-click copying features to smoothly transition generated text into external mail clients.

---

## Technologies Used

- **Vanilla JavaScript**: For logic processing, string compilation, and state mutations.
- **Tailwind CSS**: For crafting a premium, scannable, and modern responsive user interface layout.
- **HTML5**: For standard web document object semantic structures.
